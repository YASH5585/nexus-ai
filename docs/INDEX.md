## 📖 Documentation Standards

All documentation in this project follows these standards:

- **Markdown**: All documentation is written in Markdown format
- **Code Examples**: All code examples include syntax highlighting and comments
- **Diagrams**: Architecture diagrams use Mermaid syntax when possible
- **Links**: All external links open in new tabs with `target="_blank"`
- **Headers**: Use ATX-style headers (`#`, `##`, `###`) consistently
